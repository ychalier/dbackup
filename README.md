# django-dbackup

## Getting Started

### Prerequisites

You'll need Python>=3.6 (in order to support Django 3.0.2).

### Installation

#### 1. Install the application

```bash
git clone https://
pip install -r requirements.txt
```

#### 2. Integrate it into the Django project

 1. Add it to `INTALLED_APPS`
 2. Migrate the database
